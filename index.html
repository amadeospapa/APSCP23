Binary Search Project
"

In the binary search project, we heavily implemented the concept of pair programming. Pair programming is an agile programming concept that involves collaborators swapping places coding and "navigating" (i.e. directing, double-checking code, etc.) to enable a more efficient programming process. For the project, we learned about the concept of binary search, and used if statements and boolean logic to create a binary search-based algorithm that could determine an NFL team that the user thought of. Throughout the process, we implemented various methods of testing the program with various inputs.
